# Портфолио — Никита Проксет

Лендинг-портфолио: senior frontend разработчик. Лендинги, сайты и интернет-магазины под ключ.

Статический сайт (HTML / CSS / JS, без сборки).

## Возможности
- Hero с фото и описанием
- Калькулятор стоимости с подсказками
- Каталог работ (живые интерактивные превью)
- Форма-бриф с отправкой в Telegram

## Запуск локально
Откройте `index.html` в браузере или поднимите статический сервер:

```bash
npx serve .
```
